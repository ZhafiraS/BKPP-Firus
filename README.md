## Kalkulator Biaya Parkir Otomatis
Program untuk menghitung total biaya parkir mobil dan motor. Terdapat tarif dasar dan tarif per jam berikutnya. Input adalah jenis kendaraan dan durasi parkir (jam dan menit). Wajib handling error jika durasi parkir negatif.
